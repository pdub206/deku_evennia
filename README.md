# DEKU MUD (Dungeons, Encounters, Kingdoms, and Universes)

DEKU is a sapling grown from a DIKU MUD seed, which will eventually become a full blown tree of its own.

Written in Python with Evennia as a base system, I am trying to honor the legacy of DIKU and carry it forward in a new era.

## Design philosophy

DEKU is a step forward from older MUDs which had archaic combat, exploration, magic, and roleplay systems. It uses modern systems and is not afraid to experiment.

It also takes into consideration the difficulty for novice users that want to get started with running an Evennia game, streamlining it with easy to follow scripts.

We welcome issues to be created and discussion about its evolution.

### Setup

To run the game, start with:

1. ./setup.sh
2. ./game_ctl.sh start
3. Login to the game using telnet on port 4000
